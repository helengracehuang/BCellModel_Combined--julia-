340
341 `@..` turns `expr` into a broadcast-like expression, similar to `@.`.
342 It additionally provides two optional keyword arguments:
343
344 - thread: Defaults to `false`. Use multithreading?
345 - broadcast: Defaults to `false`. If `true`, it will broadcast axes with dynamic

349 be broadcast. Note that this differs from base broadcasting, in that
