49 ODEIntegrator
50 Fundamental `struct` allowing interactively stepping through the numerical solving of a differential equation.
51 The full documentation is hosted here:
52 [https://diffeq.sciml.ai/latest/basics/integrator/](https://diffeq.sciml.ai/latest/basics/integrator/).
53 This docstring describes basic functionality only!
54
55 Initialize using `integrator = init(prob::ODEProblem, alg; kwargs...)`. The keyword args which are accepted are the same
56 [common solver options](https://diffeq.sciml.ai/latest/basics/common_solver_opts/)
57 used by `solve`.
58
59
60 For reference, relevant fields of the `ODEIntegrator` are:
61
62 * `t` - time of the proposed step
63 * `u` - value at the proposed step
64 * `opts` - common solver options
65 * `alg` - the algorithm associated with the solution
66 * `f` - the function being solved
67 * `sol` - the current state of the solution
68 * `tprev` - the last timepoint
69 * `uprev` - the value at the last timepoint
70
71 `opts` holds all of the common solver options, and can be mutated to change the solver characteristics.
72 For example, to modify the absolute tolerance for the future timesteps, one can do:
73 ```julia
74 integrator.opts.abstol = 1e-9
75 ```
76 For more info see the linked documentation page.
