139 #=
140 Dealing with discontinuities
141
142 If we hit a discontinuity (this is checked in `apply_step!`), then we remove the
143 discontinuity, additional discontinuities at the current time point (if present), and
144 maybe also discontinuities and time stops coming shortly after the current time point
145 in `handle_discontinuities!`. The order of the discontinuity at the current time point is
146 defined as the lowest order of all these discontinuities.
147
148 If the problem is not neutral, we will only add additional discontinuities if
149 this order is less or equal to the order of the algorithm in
150 `add_next_discontinuities!`. If we add discontinuities, we add discontinuities
151 of the next order caused by constant lags (these we can calculate explicitly and
152 just add them to `d_discontinuities` and `tstops`) and we add the current
153 discontinuity to `tracked_discontinuities` which is the array of old
154 discontinuities that are checked by a `DiscontinuityCallback` (if existent).
155 =#

198 """
199 add_next_discontinuities!(integrator::DDEIntegrator, order[, t=integrator.t])
200
201 Add discontinuities of next order that are propagated from discontinuity of
202 order `order` at time `t` in `integrator`, but only if `order` is less or equal
203 than the order of the applied method or the problem is neutral.
204
205 Discontinuities caused by constant delays are immediately calculated, and
206 discontinuities caused by dependent delays are tracked by a callback.
207 """
208 function add_next_discontinuities!(integrator, order, t=integrator.t)
209 neutral = integrator.sol.prob.neutral
210 next_order = neutral ? order : order + 1
211
212 # only track discontinuities up to order of the applied method
213 alg_maximum_order = OrdinaryDiffEq.alg_maximum_order(integrator.alg)
214 next_order <= alg_maximum_order + 1 || return
215
216 # discontinuities caused by constant lags
217 if has_constant_lags(integrator)
218 constant_lags = integrator.sol.prob.constant_lags
219 maxlag = integrator.tdir * (integrator.sol.prob.tspan[end] - t)
220
221 for lag in constant_lags
222 if integrator.tdir * lag < maxlag
223 # calculate discontinuity and add it to heap of discontinuities and time stops
224 d = Discontinuity(integrator.tdir * (t + lag), next_order)
225 push!(integrator.d_discontinuities_propagated, d)
226 push!(integrator.tstops_propagated, d.t)
227 end
228 end
229 end
230
231 # track propagated discontinuities with callback
